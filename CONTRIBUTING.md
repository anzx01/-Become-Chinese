# Contributing to Become Chinese

Thank you for your interest in contributing! This document outlines how you can help improve this project.

## Code of Conduct

By participating in this project, you agree to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).

## How to Contribute

### Reporting Issues

- Check existing issues first to avoid duplicates
- Describe the problem clearly with steps to reproduce
- Include relevant environment details (OS, Node version, etc.)
- Remove any sensitive information before submitting

### Submitting Changes

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes following the project's coding standards
4. Test your changes thoroughly
5. Commit with clear, descriptive messages
6. Push to your fork and submit a pull request

### Pull Request Guidelines

- Keep PRs focused on a single feature or fix
- Update tests and documentation as needed
- Follow the existing code style
- Reference relevant issues in the PR description
- Be open to feedback and iterate as needed

## Content Contributions

For content updates to learning modules:

- Ensure accuracy and currency of information
- Verify with official sources for critical details
- Include disclaimers where appropriate
- Be respectful of cultural context
- Add clear attributions for any sourced material

## Development Setup

```bash
# Install dependencies
pnpm install

# Run development server
pnpm run dev

# Run linting
pnpm run lint

# Build for production
pnpm run build
```
